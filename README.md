# S3 Manager

Manage S3 compatible bucket. View, Upload & Delete.

## Setup

Add a .env file with following contents

```sh
S3_AccessKeyId="your_access_key"
S3_SecretAccessKey="your_secret_key"
S3_Bucket="your_target_bucket_name"
S3_Region="us-east-1"
S3_Endpoint="https://s3.us-east-1.amazonaws.com"
```

## Running

```sh
git clone
npm install
npm run dev

```
